## 📖 Cómo funciona

1. Escaneas un **QR** o entras al link.  
2. Aparece un botón **"Ver pista"** → cada clic muestra una pista nueva.  
3. Al final, puedes pulsar **"Revelar el libro"** y aparece el título con su enlace.
